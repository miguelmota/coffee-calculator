# Coffee Calculator

> Coffee brew calculator written in [ClojureScript](https://clojurescript.org/).

## Demo

[https://lab.miguelmota.com/coffee-calculator/](https://lab.miguelmota.com/coffee-calculator/)

## Development

Server

```sh
make serve
```

Watch

```sh
make watch
```

Lint

```sh
make lint
```

Build

```sh
make build
```
